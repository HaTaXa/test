Если сравнивать логику с VBA, то там отличие примерно такое:
а) когда нужно перебрать каждую единицу в каком-либо объекте, например:
    For i = LBound(arr) To UBound(arr)
        For j = LBound(arr, 2) To UBound(arr, 2)
            .Cells(j + 1, i + 1).Value = arr(i, j)
        Next j
    Next i
б) когда нужно для каждого объекта в объекте/коллекции... пройтись циклом, например:
    For Each rng In obj.Range
        ReDim Preserve arr(0 To 1, 0 To j) 'расширяем размер массива с сохранением предыдущих значений
        ...
        arr(0, j) = rng.Value
        j = j + 1
        ...
    Next
    ...
    Set rng = Nothing 'удалить объект из оп.памяти
    Erase arr 'очищаем сожержимое массива
