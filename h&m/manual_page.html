Итоговая аттестационная работа.

По результатам освоения программы обучения Вам необходимо выполнить Итоговую аттестационную работу на тему «Разработка web-приложения с использованием технологий HTML, CSS, JavaScript, JQuery».

Задача итоговой аттестационной работы: разработка веб-ресурса, используя гипертекстовый язык разметки HTML, каскадные таблицы стилей CSS, язык программирования JavaScript (по желанию библиотеку React).

Данное веб-приложение может являться как веб-сайтом, так и функциональным модулем (калькулятор, банковский калькулятор, форма бронирования билетов, форма обратной связи и т.д.).

Требования к выполнению: Веб-приложение должно быть создано с использованием HTML5, CSS3, JS, а также при необходимости могут использоваться библиотеки JS (jQuery, React). Интерфейс веб-приложения (веб-сайта) должен иметь адаптивный дизайн (различное визуальное представление для мобильных и десктоп устройств). Веб-приложение должно быть представлено в виде архива с набором всех необходимых файлов .html, .css, .js. Также необходимо, чтобы проект был выложен в собственный репозиторий на сервисе Github, и была представлена ссылка на него.

Требования к оформлению: итоговую аттестационную работу (разработанный проект веб-ресурса) представить в виде файлов:

  • в формате *.pdf. Файл должен содержать титульный лист с ФИО слушателя и названием проекта, описание целей и задач проекта, скриншоты всех страниц разработанного web-приложения и т.д. (в разделе размещен файл с методическими рекомендациями по выполнению работы). Файл необходимо назвать: Итоговая работа_ и Вашу фамилию и инициалы. Например: Итоговая работа_Иванов И.И.
  • в формате архива (*.rar, *.zip). Файл должен быть загружен на dropmefiles.com. Файл с архивом необходимо назвать: Архив_ и Вашу фамилию и инициалы. Например: Архив_Иванов И.И.
  • в формате *.doc. Файл должен содержать ссылку на репозиторий gitHub, где расположены файлы проекта. Причем, файл README.md находящийся в репозитории, должен включать описание проекта. Файл с ссылкой на архив итоговой аттестационной работы необходимо назвать: Ссылка_ и Вашу фамилию и инициалы. Например: Ссылка_Иванов И.И.
